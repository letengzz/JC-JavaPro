## 面向对象的思想概述

类(Class)和对象(Object)是面向对象的核心概念。  类是对一类事物的描述，是抽象的、概念上的定义  对象是实际存在的该类事物的每个个体，因而也称为实例(instance)。  “万事万物皆对象”

类的成员

 属 性：对应类中的成员变量  行 为：对应类中的成员方法

Field = 属性 = 成员变量，Method = (成员)方法 = 函数
